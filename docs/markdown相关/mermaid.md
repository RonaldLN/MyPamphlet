## 1

有标点或特殊符号，如`“` 、`<`、`（`、空格、html换行符 等时，需要用字母加`[]`来形成单元块，如：

```html
d["（“小”对“大”，“弱”对“强”）<br \>奋斗抗争的过程"]
```

## 2

支持html

-   换行  `<br \>`
-   斜体 `<i>...</i>`
-   加粗、下划线、大号小号字体 ...

subgraph g[" "]

end
